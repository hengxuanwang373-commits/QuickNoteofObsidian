# QuickNote ✏️

macOS 菜单栏备忘录，支持粘贴图片，直接保存到 Obsidian QuickNotes 文件夹。

---

## 安装完成

**App 已安装**: `/Applications/QuickNote.app`

---

## 菜单栏功能

点击顶部菜单栏 📝 图标:

| 菜单项 | 功能 |
|--------|------|
| **快速记录 ✏️** | 打开输入窗口记录笔记 |
| **设置 ⚙️** | 打开设置窗口 |
| **打开 QuickNotes** | 在 Finder 中打开笔记文件夹 |
| **退出** | 关闭应用 |

---

## 功能

### 每日文件存储
- 每天使用一个文件 `YYMMDD.md`
- 每次记录追加到当天的文件

### 图片和附件
- ✅ **⌘V 粘贴图片** - 复制图片后保存时自动上传
- ✅ 支持添加文件作为附件

---

## 设置

点击菜单栏 📝 → **设置 ⚙️** 可配置:

| 设置项 | 说明 |
|--------|------|
| **保存路径** | 选择笔记保存位置 |
| **通知** | 开关保存成功通知 |

**快捷键设置:**
设置 → 键盘 → 键盘快捷键 → 服务 → 找到 QuickNote 设置

---

## 使用方法

### 快速记录

1. **复制图片** (⌘C) - 从任何应用复制截图或图片
2. 点击菜单栏 📝 → **快速记录 ✏️**
3. 输入文字内容
4. 点击 **保存** - 剪贴板中的图片会自动上传

### 添加附件

1. 在对话框中点击 **添加附件**
2. 选择文件
3. 点击 **完成** 保存

---

## 保存格式

```markdown
## 📝 10:30 (日)

文字内容...

![image.png](attachments/103045_abc123.png)
```

---

## 登录项设置 (开机自启)

1. 打开 **系统设置** → **用户与群组** → **登录项**
2. 点击 `+` 添加 `/Applications/QuickNote.app`
